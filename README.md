# Telegram-бот для обезличивания документов

Бот автоматически обезличивает юридические документы, удаляя персональные данные.

## Возможности

- Обработка PDF и DOCX файлов
- Обезличивание:
  - ФИО и инициалов
  - Названий организаций
  - ИНН, ОГРН, ОГРНИП, КПП
  - Банковских реквизитов (БИК, расчетные и корреспондентские счета)
  - Телефонов и email
  - Паспортных данных и СНИЛС
  - Адресов

## Установка

1. Установите Python 3.8 или выше

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте бота через @BotFather в Telegram и получите токен

4. Откройте файл `telegram_anonymizer_bot.py` и замените `YOUR_BOT_TOKEN_HERE` на ваш токен:
```python
BOT_TOKEN = 'ваш_токен_здесь'
```

## Запуск

```bash
python telegram_anonymizer_bot.py
```

## Использование

1. Найдите вашего бота в Telegram
2. Отправьте команду `/start`
3. Загрузите документ (PDF или DOCX)
4. Получите обезличенный документ в формате DOCX

## Команды

- `/start` - приветствие и информация о боте
- `/help` - справка по использованию

## Технологии

- **python-telegram-bot** - взаимодействие с Telegram API
- **PyPDF2** - работа с PDF файлами
- **python-docx** - работа с DOCX файлами
- **pdf2docx** - конвертация PDF в DOCX
- **Natasha** - распознавание именованных сущностей (NER) для русского языка

## Безопасность

Все загруженные файлы автоматически удаляются после обработки. Данные не сохраняются на сервере.

## Ограничения

- Максимальный размер файла: 20 МБ (ограничение Telegram)
- PDF файлы конвертируются в DOCX
- Качество распознавания зависит от структуры исходного документа

## Важно

⚠️ Всегда проверяйте результат обезличивания вручную перед публикацией документа!

Бот использует автоматическое распознавание и может пропустить некоторые персональные данные в сложных случаях.
